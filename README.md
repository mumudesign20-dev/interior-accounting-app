# 室內設計專案帳務 App

這個資料夾可直接用 GitHub Pages 發佈。

## 發佈步驟

1. 在 GitHub 建立一個新的 repository，例如 `interior-accounting-app`。
2. 將本資料夾內的檔案全部上傳到 repository 根目錄。
3. 到 repository 的 `Settings` -> `Pages`。
4. `Build and deployment` 選擇 `Deploy from a branch`。
5. Branch 選 `main`，資料夾選 `/root`。
6. 儲存後等待 1-3 分鐘，即可取得 GitHub Pages 網址。

## 手機使用

- iPhone：用 Safari 開 GitHub Pages 網址，點分享，選「加入主畫面」。
- Android：用 Chrome 開 GitHub Pages 網址，選「安裝應用程式」或「加到主畫面」。

## 注意

目前資料存在各設備自己的瀏覽器內，手機與電腦可使用同一套介面，但尚未做多人資料庫同步。
